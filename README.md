# 하연 성장 대시보드

GitHub Pages용 정적 성장 기록 대시보드입니다.

## 업데이트 방식

형빈이 Telegram에 예를 들어 아래처럼 보내면 Hermes가 `data.json`에 추가하고 GitHub Pages에 반영합니다.

```text
하연 키 90.1cm 몸무게 14.0kg
```

## 데이터 원칙

- 공개 Pages 기준이라 이름은 `하연`만 사용합니다.
- 생년월일, 병원명, 주민정보 등 민감정보는 공개 `data.json`에 넣지 않습니다.
- 생년월일은 로컬 `.private_child.json`에만 보관하고, 공개 데이터에는 계산된 월령과 백분위만 저장합니다.
- 백분위는 질병관리청 2017 소아청소년 성장도표 LMS 기준을 사용합니다. 세브란스 어린이병원 백분위 안내도 같은 2017 성장도표를 기준으로 설명합니다.
- `percentile_curves.json`은 여아 24~60개월 기준 3/10/25/50/75/90/97백분위 곡선을 담고, 공개 페이지에서 키·몸무게·BMI별로 현재 위치를 표시합니다.

## 로컬 확인

```bash
python3 -m http.server 8000
open http://127.0.0.1:8000
```
