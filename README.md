# 하연 성장 대시보드

GitHub Pages용 정적 성장 기록 대시보드입니다.

## 업데이트 방식

형빈이 Telegram에 예를 들어 아래처럼 보내면 Hermes가 `data.json`에 추가하고 GitHub Pages에 반영합니다.

```text
하연 키 90.1cm 몸무게 14.0kg
```

## 데이터 원칙

- 공개 Pages 기준이라 이름은 `하연`만 사용합니다.
- 생년월일, 병원명, 주민정보 등 민감정보는 넣지 않습니다.
- 백분위/성장곡선 분석은 월령/성별 기준표가 들어온 뒤 추가합니다.

## 로컬 확인

```bash
python3 -m http.server 8000
open http://127.0.0.1:8000
```
